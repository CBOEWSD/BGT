Echo Class

Makes use of the echojs library to only load images when they are in the viewport

Example:

                                
                                    
                                        <
                                            img
                                            src=
                                            "/ui/assets/images/placeholders/blank.gif"
                                            data-echo=
                                            "
                                        
                                    
                                    {{
                                        src}}
                                    
                                        
                                            ">
                                    
                                
                            
(
                            
                                function
                                (){
                            Echo.init({ offset:
                            200, throttle:
                            300
                            });

In the event of a resize we trigger Echo to poll images for inview. We need this due to Echo's default listener only being from scroll events on the window. Scroll is not fired on resize.

PubSub.subscribe(
                            'resize',
                            
                                function
                                (){
                            returnEcho.render(); }); PubSub.subscribe(
                            'DomChange',
                            
                                function
                                (){
                            returnEcho.render(); }); }).call(
                            this);
