PublishResize

Allows for modules or classes to subscribe for a window resize event to be fired. This does a number of things. Primarily, it allows for periodic checking and firing as opposed to continuous firing. The event that subscriberrs are listening for will only happen once a second and prevent a performance issue for repeated adustments.
