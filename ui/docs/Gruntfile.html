    Server

    This is used to directly run the server without monitoring for changes. This is useful for quick runs - such as testing.

      grunt.registerTask 'server', 'Start our local web server.', ->
    grunt.log.writeln "Started server on port #{pkg.server.port}"
    require('./app.coffee')
